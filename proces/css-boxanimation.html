CSS Box Animation

Se hjemmeside

Hjemmeside kodet i HTML og CSS

PROCES

Til denne opgave var målet at lære at animere med keyframes i CSS. Billedet bliver roteret på tre forskellige akser i CSS med rotateX, rotateY og rotateZ. Deruover er bugt translateX og translateY for at rykke på billedet, samt width og height for at ændre på dets størrelse. Det hele er animeret efter procentmessige keyframes hvoraf der er 5. 0%, 25%, 50%, 75% og 100%. Animationen slutter på 100% i samme stadie som den er i ved 0& for at skabe et perfekt loop.

KRITISK REFLEKTION

Det var fedt at finde ud af at man rotere omkring 3 akser i CSS, og dermed lave 3-dimensionelle effekter hvis man bruger dte rigitgt. Det føler jeg ikke helt jeg gjorde til dette projekt, men jeg fik dog styr på at bruge selve funktionen, og blev bevidst om den.
